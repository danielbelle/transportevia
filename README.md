# Transporte

Projeto base para aplicação web utilizando Laravel (backend PHP) e Node.js (frontend ou utilitários JS).

## Estrutura do Projeto

```
transporte/
├── api/
├── app/
├── bootstrap/
├── config/
├── database/
├── public/
├── resources/
├── routes/
├── storage/
├── tests/
├── package.json
├── composer.json
├── phpunit.xml
├── README.md
├── tailwind.config.js
├── vercel.json
└── vite.config.js
```

## Tecnologias

-   **Backend:** Laravel `^10.0`
-   **Frontend/Utilitários:** Node.js `^20.0.0`
-   **Gerenciador de pacotes PHP:** Composer `^2.0`
-   **Gerenciador de pacotes JS:** npm `^10.0.0`
-   **Banco de Dados:** [Neon Database](https://neon.tech/) (PostgreSQL Cloud)
-   **Deploy:** [Vercel](https://vercel.com/)

## Observações

-   O deploy será realizado na Vercel.
-   O banco de dados utilizado será o Neon Database (PostgreSQL).
-   Este README apresenta apenas o esqueleto inicial do projeto.
